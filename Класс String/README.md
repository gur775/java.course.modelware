# Практика #3

## Зачем нужны три двойные кавычки?
Три двойные кавычки (""") - это текстовые блоки (Text Blocks), появившиеся в Java 13 и окончательно в Java 15. Они упрощают работу с многострочными строками.

### Они нужны для:
 1. Удобство написания многострочного текста

 2. Сохранение форматирования (отступы, переносы строк)

 3. Повышение читаемости кода

### Особенности текстовых блоков:
Начинаются и заканчиваются тремя двойными кавычками

Перенос строки после открывающих """ не включается в строку

Отступы автоматически удаляются (минимальный общий отступ)

Поддерживают \s для сохранения пробелов и \ для отмены переноса строки
