# Практика #1

## Что произойдет в результате выполнения следующего кода?
```java
int a1[] = {1,2,3,4,5};
int a2[] = {6,7,8,9,10};
a1 = a2;
```
### Результат выполнения:


До присваивания
```
a1 = [1, 2, 3, 4, 5]
a2 = [6, 7, 8, 9, 10]
```
После a1 = a2
```
a1 = [6, 7, 8, 9, 10]
a2 = [6, 7, 8, 9, 10]
```
В Java массивы - это ссылочные типы данных.
При присваивании a1 = a2 копируется не массив,
а только ссылка на него.
Переменная a1 стала указывать на массив a2.
Исходный массив [1,2,3,4,5] остался без ссылок и будет удалён сборщиком мусора.
