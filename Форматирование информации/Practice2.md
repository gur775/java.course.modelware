# Практика #2

## Зачем нужен метод flush()?
Метод flush() используется для принудительной записи данных из буфера в целевое устройство.

## Применение метода flush():
* 1. Буферизация данных для улучшения производительности
* 2. Быстрая запись важных данных
* 3. Предотвращение потери данных при аварийном завершении
* 4. Освобождение буфера для новых данных
* 5. Синхронизация между разными потоками
