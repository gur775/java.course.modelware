# Практика #3

## Какое значение будет в переменной c после выполнения данного кода?
```java
int a = 1;
{
    int b = 2;
}
int c = a + b;
```
### Объясните полученный результат.
Будет ошибка компиляции.
Переменная ```b``` объявлена внутри локального блока ```{}```. Её область видимости ограничена этим блоком. 
При попытке использовать ```b``` вне блока возникает ошибка компиляции.

# Практика #7

### Какое значение будет в переменных b, с и d после выполнения данного кода.
```java
int a = 120;
byte b = a + 10;
byte c = (byte)(a + 10);
byte d = a + 1;
```
### Объясните полученный результат.

Переменная ```b```: Ошибка компиляции. ```a + 10``` имеет тип ```int```, а присваивается в ```byte```. Java требует явного приведения.

Переменная ```c```: Работает с явным приведением. 120 + 10 = 130. Для ```byte``` диапазон -128..127. 130 выходит за пределы → происходит переполнение: 130 - 256 = -126.

Переменная ```d```: Ошибка компиляции. Та же проблема, что и с ```b```.
